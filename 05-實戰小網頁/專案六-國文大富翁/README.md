# 專案六　國文大富翁 ⭐⭐⭐（進階・多檔）

**目標**：做一個 2–4 人輪流的大富翁教學遊戲。擲骰前進，踩到「題目格」要答國文題：**答對前進、答錯後退**，先到終點者贏。
題庫獨立成 `questions.js`——換科目改它就好。

檔案分工：`index.html`（畫面）、`styles.css`（樣式）、`app.js`（棋盤與回合邏輯）、`questions.js`（題庫）。

## 開始
```powershell
cd $HOME\Desktop
mkdir 國文大富翁
cd 國文大富翁
codex
```

## 起手 prompt（複製貼上）
```
幫我做一個「國文大富翁」教學遊戲網頁，拆成 index.html、styles.css、app.js、questions.js 四個檔案。
規則：
1. 棋盤是一條約 20 格的路線，含起點、終點、幾個「題目格」、「幸運格（前進）」、「陷阱格（後退）」。
2. 可選 2～4 位玩家，輪流按「擲骰」前進（骰子 1～6）。
3. 踩到題目格時跳出國文選擇題；答對前進 2 格、答錯後退 2 格，並顯示解析。
4. 棋盤上用不同顏色棋子顯示每位玩家位置，旁邊有回合提示和操作紀錄。
5. 先到終點的玩家獲勝，跳出勝利畫面，可再玩一局。
題庫放在 questions.js（每題：題目、四選項、正解、解析），先放 12 題國中國文題。
不要用需要架伺服器的功能（雙擊 index.html 就能玩），繁體中文。
```

> ⚠️ 題庫一樣用 `<script src="questions.js">` 載入，**不要用 `fetch`**，雙擊才打得開。

## 看成果
雙擊 `index.html`，選 2 人，輪流擲骰，踩到黃色題目格時答題，看誰先到 🏁。

## 迭代挑戰
1. `每位玩家可以自己輸入隊名`
2. `加一格「機會格」：抽到隨機事件（直接前進/回到起點等）`
3. `答對連續兩題給「再擲一次」獎勵`
4. `把題庫換成自然、社會或英文，做成跨科版本`

## 卡關排解
- **棋子沒移動？** 跟 Codex 說「擲骰後玩家位置沒更新，幫我檢查 move 與重繪」。
- **答完沒換人？** 說「答完題、套用前進/後退後，要輪到下一位玩家」。
- **題目重複太快？** 說「讓題目先洗牌再輪流出，盡量不重複」。
