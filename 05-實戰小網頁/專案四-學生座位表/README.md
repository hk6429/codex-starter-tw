# 專案四　學生座位表 ⭐⭐（進階・多檔）

**目標**：做一個排座位的小工具，輸入名單就自動排好，還能**拖拉交換**、隨機洗座、列印。
這是**進階版**：跟前三個專案不同，它把程式拆成 `index.html`（畫面）、`styles.css`（樣式）、`app.js`（邏輯）三個檔案——更像真實的網頁專案。

## 開始
```powershell
cd $HOME\Desktop
mkdir 座位表
cd 座位表
codex
```

## 起手 prompt（複製貼上）
```
幫我做一個「學生座位表」網頁，拆成 index.html、styles.css、app.js 三個檔案（不要全部塞一個檔）。
功能：
1. 可輸入班級名稱、每排幾人、共幾排，和一份「一行一個名字」的學生名單。
2. 按「依名單排座」會把名字依序填進座位格子，上方有一塊「講台」當方向參考。
3. 座位上的名字可以用滑鼠互相拖拉交換。
4. 有「隨機洗座」「列印」「清空」按鈕。
5. 所有資料自動存進瀏覽器 localStorage，下次打開還在。
介面乾淨、適合老師上課用，繁體中文。
```

## 看成果
雙擊 `index.html`，輸入幾個名字、按「依名單排座」，再試著把兩個名字拖來拖去交換。

## 迭代挑戰
1. `座位格子加上座號（左上角顯示 1、2、3…）`
2. `加一個「男女分色」開關，名字後面標 (男)/(女) 就用不同底色`
3. `加「分組」功能：每 4 個座位框成一組、顯示第幾組`
4. `列印時自動隱藏所有按鈕，只留座位表和班級名稱`

## 卡關排解
- **拖不動？** 跟 Codex 說「座位上的名字拖不動，幫我檢查 draggable 和 drop 事件」。
- **改了排數座位亂掉？** 說「改變行列數時，已經排好的名字要盡量保留」。
- **想匯入 Excel 名單？** 說「讓我可以一次貼上一整欄複製來的名字（含空白行也能處理）」。
