# 專案一　自我介紹網頁 ⭐

**目標**：學會核心循環 ——「下一句話 → Codex 生出整個檔案 → 在瀏覽器打開 → 用講的修改」。

## 開始
```powershell
cd $HOME\Desktop
mkdir 自我介紹
cd 自我介紹
codex
```

## 起手 prompt（複製貼上）
```
幫我做一個單頁的個人自我介紹網頁 index.html，要有我的名字、職業（國小老師）、
三個興趣、一句座右銘，配色溫暖、字體大、手機上也好看，全部寫在一個檔案裡。
```
按同意，等它做完。

## 看成果
打開「自我介紹」資料夾，雙擊 `index.html`，用瀏覽器看你的網頁。

## 迭代挑戰（用講的改，不用改程式）
一句一句丟給 Codex：
1. `把背景改成淺綠色漸層`
2. `加一個按鈕，按了會跳出一句鼓勵的話`
3. `最上面留一個放大頭貼的位置，先用佔位圖`
4. `把我的名字改大、置中，加一點陰影`

## 卡關排解
- **找不到檔案？** 確認你是在「自我介紹」資料夾裡啟動 codex 的。
- **改錯方向？** 直接說「還原剛剛的修改」或「不要剛才那個效果，改成…」。
- **網頁打不開？** 用滑鼠右鍵 → 開啟檔案 → 選 Chrome / Edge。
