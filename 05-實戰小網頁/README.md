# 05 實戰小網頁

三個專案，難度遞增，全部對老師有用。每個專案資料夾裡有：**起手 prompt（直接複製貼給 Codex）、預期成果、迭代挑戰、卡關排解**。

| 專案 | 難度 | 你會做出 |
|---|---|---|
| [專案一 自我介紹網頁](專案一-自我介紹網頁/) | ⭐ | 一頁個人介紹網頁 |
| [專案二 課堂抽籤點名器](專案二-課堂抽籤點名器/) | ⭐⭐ | 會隨機點名／分組的互動網頁 |
| [專案三 小考測驗網頁](專案三-小考測驗網頁/) | ⭐⭐⭐ | 會自動計分的選擇題小考 |

## 共同流程
每個專案都是同一套節奏（跟 [03 第一次對話](../03-第一次對話/) 一樣）：
```powershell
cd $HOME\Desktop
mkdir 專案資料夾名
cd 專案資料夾名
codex
```
然後把該專案的「起手 prompt」貼進去，按同意，做完用瀏覽器打開產生的 `.html` 檔看成果。

> ⚠️ 上面的 **`專案資料夾名` 只是代稱**，不要照字面打。實際要用各專案頁給你的名字（例如「自我介紹」「抽籤點名器」「小考測驗」）——每個專案頁的開頭都已經幫你寫好整段可直接複製的指令了，照那個用最快。

> 💡 **多行 prompt 怎麼貼**：起手 prompt 有好幾行，整段一次複製、在 Codex 輸入框按右鍵（或 Ctrl+V）貼上，**全部貼完再按 Enter 送出**。如果貼到一半就自己送出去了，不用緊張——直接把剩下那幾行再貼一次補上即可。
>
> 走 [App 版](../01b-用App版/) 的人：把 `cd`／`mkdir` 那幾行指令略過，改成**在 App 裡新增／選擇對應資料夾**，再把起手 prompt 貼進對話框就好。

## 課堂節奏建議
- 專案一：**全班一起做**，老師示範「下指令 → 同意 → 看成果 → 用講的修改」的循環。
- 專案二：**自己動手**，老師巡堂。
- 專案三：**挑戰題**，時間不夠可當回家作業。
