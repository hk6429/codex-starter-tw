### 進階專題（多檔版・行有餘力再做）

前三個專案是「全部寫在一個檔案」的入門練習。下面三個改用**多檔結構**（`index.html` + `styles.css` + `app.js`，題庫再獨立成 `questions.js`）——更接近真實網頁專案的長相，也更實用。每個資料夾裡都附了**可直接雙擊執行的完整成品**，可當範例參考或起手 prompt 的對照答案。

| 專案 | 難度 | 你會做出 |
|---|---|---|
| [專案四 學生座位表](專案四-學生座位表/) | ⭐⭐ | 輸入名單自動排座、可拖拉交換、列印 |
| [專案五 國文複習刷題](專案五-國文複習刷題/) | ⭐⭐⭐ | 即時對答案＋解析、可只重練答錯題 |
| [專案六 國文大富翁](專案六-國文大富翁/) | ⭐⭐⭐ | 2–4 人輪流擲骰、踩格答題的教學遊戲 |

> 💡 這三個的題庫都放在獨立的 `questions.js`——**換科目只要改那個檔，不必動程式**。為了能雙擊就打開，題庫一律用 `<script>` 載入，不用需要架伺服器的 `fetch`。

## 共同流程
每個專案都是同一套節奏（跟 [03 第一次對話](../03-第一次對話/) 一樣）：
```powershell
cd $HOME\Desktop
mkdir 專案資料夾名
cd 專案資料夾名
codex
```
然後把該專案的「起手 prompt」貼進去，按同意，做完用瀏覽器打開產生的 `.html` 檔看成果。

> ⚠️ 上面的 **`專案資料夾名` 只是代稱**，不要照字面打。實際要用各專案頁給你的名字（例如「自我介紹」「抽籤點名器」「小考測驗」）——每個專案頁的開頭都已經幫你寫好整段可直接複製的指令了，照那個用最快。

> 💡 **多行 prompt 怎麼貼**：起手 prompt 有好幾行，整段一次複製、在 Codex 輸入框按右鍵（或 Ctrl+V）貼上，**全部貼完再按 Enter 送出**。如果貼到一半就自己送出去了，不用緊張——直接把剩下那幾行再貼一次補上即可。
>
> 走 [App 版](../01b-用App版/) 的人：把 `cd`／`mkdir` 那幾行指令略過，改成**在 App 裡新增／選擇對應資料夾**，再把起手 prompt 貼進對話框就好。

## 課堂節奏建議

- 專案二：**自己動手**，老師巡堂。
- 專案三：**挑戰題**，時間不夠可當回家作業。
