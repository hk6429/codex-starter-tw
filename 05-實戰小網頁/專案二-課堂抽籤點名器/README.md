# 專案二　課堂抽籤點名器 ⭐⭐

**目標**：做出第一個「會動」的互動網頁，當天上課就能用。

## 開始
```powershell
cd $HOME\Desktop
mkdir 抽籤點名器
cd 抽籤點名器
codex
```

## 起手 prompt（複製貼上）
```
幫我做一個課堂隨機點名網頁 picker.html。我可以在文字框輸入全班同學名字
（一行一個），按「抽一位」會隨機選出一個名字並用大字顯示；再做一個「分成幾組」
功能，輸入組數就把全班隨機平均分組顯示出來。介面簡單可愛，全部寫在一個檔案。
```

## 看成果
雙擊 `picker.html`，貼上幾個名字試抽看看。

## 迭代挑戰
1. `抽過的人不要再被抽到，並在下面顯示已抽名單`
2. `抽中名字時加一個放大跳動的小動畫`
3. `加一個「重設」按鈕，可以清空已抽名單`
4. `分組時每組用不同顏色的卡片顯示`

## 為什麼這個值得學
這就是一個真正的小工具：有輸入、有按鈕、有隨機邏輯。但你全程只用中文下指令，沒寫半行程式。

## 卡關排解
- **按鈕沒反應？** 跟 Codex 說「按鈕點了沒反應，幫我修好」。
- **想要更大字／更好看？** 直接描述：「名字再大一倍、用圓潤字體」。
