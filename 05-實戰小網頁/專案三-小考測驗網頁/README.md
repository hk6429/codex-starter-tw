# 專案三　小考測驗網頁 ⭐⭐⭐

**目標**：做一個有題目、會自動計分的小程式，並拿它完整跑一次「推上 GitHub」。

## 開始
```powershell
cd $HOME\Desktop
mkdir 小考測驗
cd 小考測驗
codex
```

## 起手 prompt（複製貼上）
```
幫我做一個五題的選擇題小考網頁 quiz.html，題目主題是「網路與 3C 使用安全」，
每題四個選項，作答完按「看結果」會顯示分數，並標出哪幾題答錯、正確答案是什麼。
介面清楚、適合中小學生看，全部寫在一個檔案。
```
（注意：題目讓 Codex 自己生，你只要審一下對不對。）

## 看成果
雙擊 `quiz.html`，自己作答一次，看計分對不對。

## 迭代挑戰
1. `題目改成 10 題，每題加上一句解析`
2. `答錯的選項標紅色、答對標綠色`
3. `最後加一句：80 分以上顯示「你是網路安全小高手！」`
4. `主題換成我自己的科目（例如國語生字、數學分數）`

## 接著：推上 GitHub
這個成品最適合拿來練習發布。做完直接前往 [06 串接 GitHub](../../06-串接GitHub/) → [07 推上 GitHub](../../07-推上GitHub/)。

## 卡關排解
- **計分錯誤？** 跟 Codex 說「我全對卻只顯示 4 分，幫我檢查計分」。
- **想印出來當紙本考卷？** 說「加一個列印按鈕，列印時不要顯示按鈕」。
