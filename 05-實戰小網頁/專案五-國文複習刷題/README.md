# 專案五　國文複習刷題 ⭐⭐⭐（進階・多檔）

**目標**：做一個會出題、即時對答案、給解析、最後能**只重練答錯題**的刷題網站。
題庫獨立成一個 `questions.js`——**換科目只要改這個檔，不必動程式**。

檔案分工：`index.html`（畫面）、`styles.css`（樣式）、`app.js`（流程）、`questions.js`（題庫）。

## 開始
```powershell
cd $HOME\Desktop
mkdir 國文刷題
cd 國文刷題
codex
```

## 起手 prompt（複製貼上）
```
幫我做一個「國文複習刷題」網頁，拆成 index.html、styles.css、app.js、questions.js 四個檔案。
要求：
1. 題庫放在 questions.js，是一個陣列，每題有「題目、四個選項、正解、解析、分類標籤」。
   先幫我放 10 題國中國文題（字音、字形、成語、修辭、國學常識都來一些）。
2. 一次顯示一題，點選項後立刻顯示對錯、標出正解、顯示解析，再按「下一題」。
3. 上方有進度條和目前答對數，可選擇題目隨機出現。
4. 全部做完顯示分數與評語，並提供「只重練答錯的」按鈕。
5. 用 localStorage 記住最佳分數。
介面適合中小學生、繁體中文，不要用到需要架伺服器的功能（雙擊就能開）。
```

> ⚠️ 重點：請 Codex **不要用 `fetch` 讀 JSON**（那需要架伺服器）。題庫直接寫成 `questions.js` 裡的變數，用 `<script>` 載入，這樣雙擊 `index.html` 就能跑。

## 看成果
雙擊 `index.html`，從頭刷一遍，故意答錯幾題，最後按「只重練答錯的」看看。

## 迭代挑戰
1. `把題庫擴充到 30 題，並依分類（字音/字形/成語…）讓我可以選範圍`
2. `加一個計時器，看完成全部要幾秒`
3. `答對連續 5 題給一句鼓勵彈窗`
4. `把題庫換成我自己的科目（數學、英文、社會都行）`

## 卡關排解
- **雙擊打開是空白？** 多半是用了 `fetch`。跟 Codex 說「改成用 script 載入 questions.js，不要 fetch」。
- **解析沒出現？** 說「答完要顯示該題的 explain 欄位」。
- **想自己加題？** 打開 `questions.js`，照同樣格式往陣列裡加一筆就好。
