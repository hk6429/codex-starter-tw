# 01b 用 App 版（不想碰終端機的人走這條）

> 這是 [01 安裝 Codex](../01-安裝Codex/)（指令版）的**替代路線**。
> 兩條二選一就好：怕終端機、想全程用視窗點選的人，走這條 App 版。

## App 版 vs 指令版，差在哪？

| | App 版（這頁） | 指令版（01） |
|---|---|---|
| 要裝什麼 | 一個應用程式，雙擊安裝 | Node.js + 打 npm 指令 |
| 操作方式 | **全部用視窗、滑鼠點** | 在 PowerShell 打字 |
| 同意它動手 | 按畫面上的「Approve」鈕 | 終端機裡用方向鍵選 |
| 適合 | 怕終端機、想最直覺的人 👈 | 想順便學終端機的人 |

兩條路做出來的成果一樣，後面 [05 實戰小網頁](../05-實戰小網頁/) 也通用。

---

## 步驟 1：安裝 Codex App

Windows 最簡單的方式：

1. 開「**Microsoft Store**」（開始選單搜尋 `Microsoft Store`）。
2. 在裡面搜尋 **`Codex`**，找到 OpenAI 出的那個，按**取得／安裝**。
3. 裝好後從開始選單打開 **Codex**。

> 也可以到官方頁面 https://developers.openai.com/codex/app 下載。

---

## 步驟 2：登入

打開 App 後，選 **Sign in with ChatGPT**，用你的**付費 ChatGPT 帳號（Plus 即可，不必到 Pro）**登入授權即可。一次登入，以後不用再登。

---

## 步驟 3：⚠️ 第一次用，務必選「Local（本機）」

開新對話時，App 會問你要 **Local（本機）** 還是 **Cloud（雲端）**：

- ✅ **一定選 Local（本機）**：檔案會乖乖待在你電腦的資料夾裡，做完能直接雙擊打開來看。
- ❌ **不要選 Cloud（雲端）**：那是把程式跑在網路上的進階模式，要先連 GitHub、檔案不在你桌面，新手會卡死。

接著選一個工作資料夾（例如先在桌面新增一個「codex-練習」資料夾，再選它）。

---

## 步驟 4：下你的第一個指令

在對話框直接用中文打一句話，例如：

```
建立一個 hello.txt，裡面寫「我學會用 Codex 了！」
```

送出後，Codex 會說它要建立檔案，並跳出一個確認。**按「Approve（同意）」**，它就動手了。
打開你剛剛選的資料夾，就會看到 `hello.txt`。🎉

> 📸 **【待補截圖】**：App 裡跳出確認、可以按「Approve」鈕的畫面。

> 看不懂它要做什麼時，先按「拒絕／No」，舉手問老師。

---

## 步驟 5（選用）：App 設定 — 一般

App 左下角齒輪 → **Settings（設定）→ General（一般）**，這裡能調 Codex 的預設行為。**全部照新手建議設一次就好，之後不用再動。**

| 設定 | 這是什麼 | 新手建議 |
|---|---|---|
| **工作模式** | 回應要多技術細節還是日常口吻 | 選 **日常工作**（技術細節較少，夠用） |
| **預設權限** ⚠️ | 「完整存取權」＝允許 Codex **未經同意就執行可聯網的指令**，風險高 | **先關掉**、保持「先問再過」最安全（同 [04 三種授權模式](../04-三種授權模式/)） |
| **預設開啟目的地** | 做好的檔案要用什麼開 | 選 **Default app**：系統依檔案類型自動開（網頁用瀏覽器）。進階工程師才換 Terminal／Git |
| **整合終端** | App 內建終端機用哪個 shell | 預設 **PowerShell**、語言自動偵測，照預設即可 |
| **多行提示** | 一次貼多行 prompt 的送出方式 | **建議開啟**：之後按 **Shift + Enter** 換行、Enter 才送出，避免打到一半就送出 |
| **後續跟進行為** | Agent 工作中你又傳訊息時怎麼處理 | 選 **排入序列**：等它做完當前任務再依序處理，避免打斷工作 |
| **程式碼審查** | 多個任務的審查要不要共用情境 | 選 **獨立**：不同任務各自獨立執行、避免互相干擾 |

> 💡 最關鍵的是 **預設權限**：上課全程維持「先問再過」，看清楚 Codex 要做什麼再按 Approve。熟了之後想加快，再參考 [04 三種授權模式](../04-三種授權模式/) 調整。

你已經會用 App 版的核心循環了，可以**跳過 [03][04]（那兩關是講指令版的細節）**，直接前往：

## 小提醒

- **學校公務機可能裝不了**（沒有安裝權限）。這條 App 版和指令版都一樣，建議用**自己的筆電**上課。
- 推 GitHub（[06 串接](../06-串接GitHub/) → [07 推上](../07-推上GitHub/)）：App 內建 Git 功能，也可以直接跟 Codex 說「幫我把這個資料夾推上 GitHub」，它會帶你做。
