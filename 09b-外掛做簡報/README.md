# 09b 用外掛做簡報（文件 → PPT）

> 🟡 **應用關。** 學會裝外掛（同 [07c 發佈上線](../07c-發佈上線/) 的 Plugins 面板）後，再介紹一個老師超實用的外掛：**Presentations**——丟一份文件給它，自動生出**圖文並茂的簡報**。

## 這能做什麼？
把一份**文字稿、教案、政策說明、課文重點**（放在 Google 文件或直接貼內容），交給 Codex，自動排成一份 PPT，省去一張一張做投影片的時間。

---

## 步驟 1：安裝 Presentations 外掛程式
打開 Codex 左側邊欄的 **「外掛程式（Plugins）」** 面板，找到 **Presentations**，點 **加號（＋）** 或「安裝」加入。

## 步驟 2：準備你的文件
把要做成簡報的內容整理好。最方便的是放在一份 **Google 文件**，複製它的分享連結；或直接把文字貼進對話框也行。

> 用 Google 文件連結時，記得把文件**權限設成「知道連結的人可檢視」**，Codex 才讀得到。

## 步驟 3：下提示詞
在對話框告訴 Codex 你要的簡報，並附上文件連結，例如：
```
根據以下文件內容製作一個圖文並茂的 PPT 來介紹政策：
https://docs.google.com/document/d/你的文件ID/edit
```

## 步驟 4：取得簡報
Codex 會讀完文件、規劃章節、配上版面與圖，產出一份簡報檔給你下載或開啟。覺得哪裡要改，直接用講的：
```
第 3 頁字太多，幫我精簡成 3 個重點；整體配色換成藍綠色系。
```

---

## 老師可以怎麼用
- **政策／校務宣導**：把公文或說明文件丟進去，幾分鐘生出宣導簡報。
- **課文導讀**：把課文重點整理成文件，自動變成上課用投影片。
- **研習分享**：把講稿轉成圖文簡報，再用 [07c 發佈上線](../07c-發佈上線/) 或直接分享檔案。

## 卡關排解
- **Codex 說讀不到文件？** 多半是 Google 文件權限沒開。改成「知道連結的人可檢視」再貼一次連結。
- **簡報內容跟文件對不上？** 提示詞講清楚對象與重點，例如「對象是國中生、用淺白的話、每頁不超過 5 行」。
- **想要特定頁數或風格？** 直接在提示詞指定：「做 8 頁、每頁一個重點、加上適合的示意圖、繁體中文」。
