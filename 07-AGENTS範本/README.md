# 07 AGENTS 範本

## 什麼是 AGENTS.md？
在專案資料夾放一個 `AGENTS.md`，等於給 Codex 的「常駐工作須知」——每次它在這個資料夾工作都會先看一眼，照你訂的規矩做。

不用每次重講「請用繁體中文」「全部寫在一個檔案」這類要求，寫進 `AGENTS.md` 一次就好。

## 怎麼用
1. 在專案資料夾建立一個檔案叫 `AGENTS.md`。
2. 把你常重複的要求寫進去（可直接抄下面範本）。
3. 之後 Codex 就會自動遵守。

> 偷懶法：在 codex 裡說「幫我建立一個 AGENTS.md，要求你之後一律用繁體中文台灣用語、程式盡量寫在單一檔案、改動前先說明」，它會幫你寫好。
