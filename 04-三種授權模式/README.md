# 04 三種授權模式

Codex 預設「動手前會先問你」，這是保護你的電腦。你可以依信任程度調整它的自由度。

> 先看懂兩個動作：**「改檔」**＝它要新增或修改你電腦裡的檔案；**「跑指令」**＝它要在電腦上執行一行命令（例如安裝東西、移動檔案）。下面三種模式，差別就在「這兩件事要不要每次先問過你」。

| 模式 | 它能做什麼 | 適合誰 |
|---|---|---|
| **建議模式（預設）** | 每次要改檔、跑指令都先問你同意 | 新手、最安全 👈 上課就用這個 |
| **自動改檔** | 在這個資料夾內可自己讀寫檔案，但執行指令仍會問 | 熟一點之後、想加快 |
| **全自動** | 在資料夾內可自由讀寫、執行指令，不再一直問 | 進階、明確知道風險時 |

## 怎麼切換
啟動時可指定，例如：
```powershell
codex            # 預設＝建議模式
```
進到 Codex 後，也可以用 `/` 叫出選單調整核准方式（介面上會顯示目前模式）。

> 不同版本選單文字可能略有不同，找到「approval / 核准」或「mode / 模式」相關選項即可切換。

## 給新手的安全守則 🔐
1. **上課全程用「建議模式」**，看清楚它要做什麼再按同意。
2. 看不懂它要跑的指令，就先別同意，舉手問老師。
3. 不要在放有重要資料的資料夾裡開全自動。
4. 不要把密碼、金鑰貼進對話。
