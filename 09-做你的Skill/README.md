# 09 做你的 Skill

## 什麼是 Skill？
把你**常常重複下的指令**，包裝成一個「一鍵技能」。以後不用每次打一長串，叫名字就能用。

例如：與其每次都打「幫我出 5 題五年級數學選擇題，附答案和解析，做成可列印網頁…」，
不如做一個叫 `小考產生器` 的 Skill，以後只要說「出考卷」就好。

## Skill 長什麼樣子？
一個資料夾，裡面放一個 `SKILL.md`。檔案分兩部分：

```
---
name: 技能名字
description: 描述「什麼時候該用這個技能」（讓 Codex 知道何時自動啟用）
---
（這裡寫被叫到時，Codex 要照著做的指示）
```

## 放在哪裡？
放在你個人的 Codex 資料夾，這樣**在任何資料夾都叫得到**：

```
C:\Users\你的名字\.codex\skills\技能名字\SKILL.md
```

（本課只教這個「個人用」位置，最單純。想做成跟整個專案綁在一起、可分享給同事的版本，是進階用法，之後再研究即可。）

## 三種做法
1. **手動建檔（本課教這個，最好懂）**：自己建資料夾、寫 `SKILL.md`。
2. **內建產生器**：在 codex 裡請它「幫我做一個 skill」，它會問你要做什麼、何時觸發。
3. **Record & Replay**：錄一次你的操作，Codex 自動幫你整理成 skill。

> 補充：Codex 舊的「custom prompts」已被官方標記淘汰，現在統一用 Skill。

## 怎麼叫出來用
在 codex 裡輸入 `/skills`，會跳出清單讓你挑；或直接用名字呼叫。

---

## 兩個現成範例（老師超實用）
- [`小考產生器`](examples/小考產生器/SKILL.md)：說「出考卷」就幫你生選擇題小考。
- [`家長訊息`](examples/家長訊息/SKILL.md)：把流水帳重點，改寫成得體的家長聯絡訊息。

### 怎麼安裝這兩個範例
把 `examples` 裡的資料夾複製到你的 `.codex\skills\`。在 codex 裡也可以直接說：
```
幫我把這個 repo 的 09-做你的Skill/examples 裡兩個 skill，複製到我的 ~/.codex/skills/
```

---

## 你的練習 🎯
照範本，做一個**你自己科目／班級**用得到的 skill，例如：
- `週報產生器`：把本週流水帳變成班級週報。
- `課文重點`：貼一段課文，整理成重點與提問。
- `評語小幫手`：輸入學生表現，產出成績單評語。

做好後用 `/skills` 叫出來測測看。
