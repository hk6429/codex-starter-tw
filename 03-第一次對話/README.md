# 03 第一次對話

重點觀念：**Codex 會在「你現在所在的資料夾」裡工作。** 所以先建一個乾淨的資料夾，等一下做的東西才不會散落各處。

### 步驟 1：建立並進入工作資料夾
在 PowerShell 一行一行貼上（每行貼完按 Enter）：
```powershell
cd $HOME\Desktop
mkdir codex-練習
cd codex-練習
```
這三行分別是：

- `cd $HOME\Desktop`：切換到你的**桌面**。（`cd` = change directory「換資料夾」，`$HOME` 是你的使用者主資料夾。）
- `mkdir codex-練習`：建立一個叫「codex-練習」的新資料夾。（`mkdir` = make directory「建資料夾」。）
- `cd codex-練習`：走進剛剛建好的那個資料夾。

做完後，PowerShell 開頭那行路徑結尾會變成 `...\codex-練習>`，表示你「人」已經在這個資料夾裡了。

### 步驟 2：啟動 Codex
```powershell
codex
```
按 Enter 後會看到一個歡迎畫面、最下方有一個輸入框。這就代表 Codex 開好了，**接下來不是打電腦指令，而是直接用中文「跟它說話」。**

### 步驟 3：下你的第一個指令
啟動後畫面最下方會有一個輸入框。直接用中文打一句話，例如：
```
建立一個 hello.txt，裡面寫「我學會用 Codex 了！」
```
打完按 **Enter** 送出。

接著 Codex 會說它要建立檔案，並跳出一個確認選單問你同不同意。
**用上下方向鍵選「Yes／Approve（同意）」，再按 Enter。** 它就動手了。

> 📸 **【待補截圖】**：Codex 跳出確認選單、問你要不要同意的那個畫面。

> 看不懂它要做什麼時，先選「No」不要同意，舉手問老師。

### 步驟 4：看成果
打開桌面的「codex-練習」資料夾，會看到 `hello.txt`。打開它，裡面就是那句話。

> 📸 **【待補截圖】**：桌面「codex-練習」資料夾裡出現 `hello.txt` 的畫面。

🎉 恭喜，你剛剛指揮 AI 在你的電腦上做了第一件事！

### 步驟 5：怎麼離開 Codex
做完想回到一般的 PowerShell：在輸入框打 `/quit` 按 Enter，或按一下 **Ctrl + C**。
想再進來就再輸入一次 `codex`。

## 你會發現的三件事
1. Codex 動手前會**先問你同意**（這是安全設計，下一章說明）。
2. 它看得懂中文需求。
3. 你不用懂任何程式語法。
