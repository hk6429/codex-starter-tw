# 03 第一次對話

重點觀念：**Codex 會在「你現在所在的資料夾」裡工作。** 所以先建一個乾淨的資料夾。

### 步驟 1：建立並進入工作資料夾
在 PowerShell：
```powershell
cd $HOME\Desktop
mkdir codex-練習
cd codex-練習
```
（這會在桌面建一個「codex-練習」資料夾並進去。）

### 步驟 2：啟動 Codex
```powershell
codex
```

### 步驟 3：下你的第一個指令
啟動後畫面最下方會有一個輸入框。直接用中文打一句話，例如：
```
建立一個 hello.txt，裡面寫「我學會用 Codex 了！」
```
打完按 **Enter** 送出。

接著 Codex 會說它要建立檔案，並跳出一個確認選單問你同不同意。
**用上下方向鍵選「Yes／Approve（同意）」，再按 Enter。** 它就動手了。

> 📸 **【待補截圖】**：Codex 跳出確認選單、問你要不要同意的那個畫面。

> 看不懂它要做什麼時，先選「No」不要同意，舉手問老師。

### 步驟 4：看成果
打開桌面的「codex-練習」資料夾，會看到 `hello.txt`。打開它，裡面就是那句話。

> 📸 **【待補截圖】**：桌面「codex-練習」資料夾裡出現 `hello.txt` 的畫面。

🎉 恭喜，你剛剛指揮 AI 在你的電腦上做了第一件事！

### 步驟 5：怎麼離開 Codex
做完想回到一般的 PowerShell：在輸入框打 `/quit` 按 Enter，或按一下 **Ctrl + C**。
想再進來就再輸入一次 `codex`。

---

## 你會發現的三件事
1. Codex 動手前會**先問你同意**（這是安全設計，下一章說明）。
2. 它看得懂中文需求。
3. 你不用懂任何程式語法。
